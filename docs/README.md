# Nobrake_CANoeOTA
OTA 기반 첨단 차량 기능 업데이트 시스템

## 프로젝트 소개
이 프로젝트의 목표는 클라우드 서버와 차량용 통신 시뮬레이터(CANoe)를 연동하여 OTA 프로세스를 구축하고, 시나리오에 따른 시뮬레이션를 구현하는 것입니다.

## 프로그램 구조
![아키텍쳐 설계](./docs/images/arc.png)

## 주요 기능
### 클라우드 기반 인프라
차량 Telematics에서 클라우드에 업데이트 데이터 요청
### 실시간 펌웨어 업데이트
ECU 식별 및 데이터 전송을 통한 펌웨어 업데이트
### New 소프트웨어 업데이트
기존에 없던 새로운 기능(스마트 라이트 빔 시스템) 추가
